# Diyana Nicolas Real Estate

Static React + Vite site for showcasing Sedona real estate services.

## Local development

```sh
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm install
npm run dev
```

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
